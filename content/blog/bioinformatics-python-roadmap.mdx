# 从零到 SCI：前端工程师转生信分析的 Python 学习路线

如果你的目标是**利用 Python 做生物信息学分析并最终发表 SCI 论文**，那么你需要学习的不是单纯的 Python，而是：

> **Python + 生物信息学 + 统计学 + 机器学习 + 论文写作**

对于已经有前端开发背景的人来说，编程反而是整个链条里最容易的一环。

## 生信论文的主流方向

目前发表难度从低到高排列：

| 方向 | 难度 | 发文速度 | 是否需要实验室 |
|------|------|----------|----------------|
| 公共数据库挖掘 | ⭐⭐ | 快 | 否 |
| 转录组分析 | ⭐⭐⭐ | 快 | 否 |
| 肿瘤预后模型 | ⭐⭐⭐ | 快 | 否 |
| 单细胞分析 | ⭐⭐⭐⭐ | 中 | 否 |
| 空间转录组 | ⭐⭐⭐⭐⭐ | 中 | 需数据 |
| 多组学分析 | ⭐⭐⭐⭐⭐ | 慢 | 需协作 |
| 实验验证型 | ⭐⭐⭐⭐⭐⭐ | 慢 | 是 |

**对于个人开发者，优先选择 TCGA + GEO 数据挖掘路线。** 原因很简单：不需要实验室、不需要测序、公开数据免费、Python/R 即可完成、发文案例极多。

## 必备知识体系

### 第一阶段：生物学基础

这部分对非生物背景的人来说是最陡的学习曲线，但不需要成为生物学专家——够用就行。

**分子生物学核心概念：**

- DNA → RNA → 蛋白质的中央法则
- 转录（Transcription）与翻译（Translation）的基本过程
- 基因表达调控的简单模型

推荐教材：*Molecular Biology of the Cell*（前 4 章就够了）。

**遗传学核心概念：**

- 基因、染色体、等位基因
- SNP（单核苷酸多态性）
- 体细胞突变 vs 胚系突变

**肿瘤学基础（如果做肿瘤方向）：**

- 癌症的发生机制（多步骤致癌）
- 驱动基因（Driver gene）vs 乘客基因（Passenger gene）
- 经典基因：TP53、EGFR、KRAS、MYC

你不需要背下所有信号通路，但需要理解"为什么某个基因的高表达可能和预后相关"。

### 第二阶段：Python 数据分析基础

你已经会编程，这部分会非常快。重点掌握科学计算栈：

```
numpy  →  矩阵和数组运算
pandas →  表格数据处理（生信数据的本质就是表格）
matplotlib + seaborn → 数据可视化
scipy  →  统计检验
```

**Pandas 是重中之重。** 你必须熟练：

# 分组汇总——按癌症分期统计基因表达

df_merged = pd.merge(expr_df, clinical_df, on='patient_id')

## 生信 Python 工具链

### 统计检验 — SciPy

```python
from scipy import stats

# t 检验：比较癌与癌旁的表达差异
t_stat, p_value = stats.ttest_ind(tumor_group, normal_group)

# 秩和检验：数据不服从正态分布时的备选
u_stat, p_value = stats.mannwhitneyu(tumor_group, normal_group)

# 卡方检验：分类变量关联分析
chi2, p, dof, expected = stats.chi2_contingency(contingency_table)
```

### 机器学习 — Scikit-learn

构建预后模型的核心：

```python
from sklearn.linear_model import LassoCV
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split, cross_val_score

# LASSO 特征筛选——从几万个基因中选出十几个关键基因
lasso = LassoCV(cv=5)
lasso.fit(X_train, y_train)
selected_genes = X.columns[lasso.coef_ != 0]

# 构建随机森林预后模型
rf = RandomForestClassifier()
scores = cross_val_score(rf, X, y, cv=5)
print(f'5-fold CV AUC: {scores.mean():.3f}')
```

## 获取公开数据

### TCGA — The Cancer Genome Atlas

最大的癌症基因组数据库，涵盖 33 种癌症类型，包含：

- RNA-seq 表达数据
- 临床信息（生存时间、分期、年龄、性别）
- 突变数据（WES/WGS）
- 甲基化数据

常用下载方式：

- [GDC Data Portal](https://portal.gdc.cancer.gov/) — 官方下载
- [UCSC Xena](https://xenabrowser.net/) — 预处理好的数据，推荐入门用
- [cBioPortal](https://www.cbioportal.org/) — 在线分析 + 下载

### GEO — Gene Expression Omnibus

NCBI 维护的基因表达数据库，包含芯片和 RNA-seq 数据。当 TCGA 的数据量不够时，用 GEO 数据集做外部验证。

### GTEx — Genotype-Tissue Expression

正常组织表达数据库，配合 TCGA 使用：癌组织 vs GTEx 正常组织做差异分析，样本量更大、结果更可靠。

## 第一篇论文的推荐路线

以下是一条经过验证的经典路径（以肝癌 LIHC 为例）：

### Step 1：获取数据

从 TCGA 下载 LIHC（肝细胞癌）的 RNA-seq 表达数据和临床数据。

### Step 2：选择目标基因

可以选一个已有文献支持但尚未在肝癌中充分研究的基因，例如与细胞周期相关的：

```
MYC / CDK1 / CCNB1 / BIRC5 / TOP2A
```

### Step 3：差异表达分析

```text
癌组织 vs 正常组织
↓
筛选标准：|log2FC| > 1 且 p-value < 0.05
```

### Step 4：生存分析

将患者按目标基因表达中位数分为高表达组和低表达组，绘制 Kaplan-Meier 曲线，用 log-rank 检验判断差异是否显著。

### Step 5：Cox 回归

单因素 Cox 和多因素 Cox 回归，评估该基因是否为独立预后因子，输出 HR（风险比）和 95% CI。

### Step 6：功能富集分析

对该基因的共表达基因做 GO 和 KEGG 富集分析，推测其涉及的生物学功能和信号通路。

### Step 7：构图

论文必需的图：

- **火山图**（差异表达概览）
- **热图**（差异基因表达模式）
- **KM 曲线**（生存分析可视化）
- **ROC 曲线**（预后模型评估）
- **森林图**（Cox 回归结果）

### Step 8：撰写论文

标准结构：

> Abstract → Introduction → Methods → Results → Discussion → Conclusion

## 进阶路线：单细胞分析

单细胞 RNA-seq（scRNA-seq）是目前最热门的方向，用 **Scanpy**（Python 版单细胞分析框架）：

```bash
pip install scanpy
```

核心流程：

```python
import scanpy as sc

adata = sc.read_10x_mtx('data/filtered_feature_bc_matrix/')
sc.pp.filter_cells(adata, min_genes=200)
sc.pp.filter_genes(adata, min_cells=3)
sc.pp.normalize_total(adata)
sc.pp.log1p(adata)
sc.pp.highly_variable_genes(adata)
sc.tl.pca(adata)
sc.pp.neighbors(adata)
sc.tl.umap(adata)
sc.tl.leiden(adata)          # 聚类
sc.tl.rank_genes_groups(adata, 'leiden')  # 找 Marker 基因

sc.pl.umap(adata, color=['leiden', 'CD3D', 'CD8A'])
```

单细胞文章的 IF 普遍在 3~10 分，甚至更高。

## AI + 生信（未来的方向）

你有前端开发背景，AI 方向其实更适合你：

- **基因功能预测** — 用 Transformer / BERT 建模基因序列和功能的关系
- **药物敏感性预测** — Deep Learning 预测患者对特定药物的响应
- **多组学融合** — GNN / Transformer 整合表达、突变、甲基化多维度数据

这是当前高分论文的热点，也是你作为工程师最能发挥技术优势的领域。

> **TCGA + GEO 数据挖掘 → Python 统计分析 → 生存分析 → 机器学习预后模型 → SCI 3~5 分**
